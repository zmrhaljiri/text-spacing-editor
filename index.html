Text Spacing Editor

WCAG 1.14.12 - Text Spacing - AA - no loss of content or functionality occurs by setting all of these properties to the following values:

  • Line height to at least 1.5 times the font size
  • Letter spacing to at least 0.12 times the font size
  • Word spacing to at least 0.16 times the font size
  • Spacing following paragraphs to at least 2 times the font size

Adjust text spacing properties by using controls below.

Activate the Set WCAG values button to set all of the properties to their minimum required values.

Activate the Reset values button to reset all the values to their previous values.
